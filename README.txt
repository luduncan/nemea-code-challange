- Required technologies in order to compile and run:

1. Java 8
2. Apache Maven 2 or higher
3. Firefox browser

run 'mvn clean install' in order to execute and run all tests


Additional info:
Installation guide for Java 8 - https://docs.oracle.com/javase/8/docs/technotes/guides/install/install_overview.html
Installation guide for Apache Maven - http://maven.apache.org/install.html
Installation guide for Mozilla Firefox - https://www.mozilla.org/en-US/firefox/new/
